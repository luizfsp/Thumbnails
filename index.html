Como Funciona a IA?

Nós incluímos uma tradução de demonstração perfeita de alta qualidade para o tema padrão de finanças/futebol/sucesso, para que você possa testar o aplicativo imediatamente.

Ao inserir sua Chave de API do Gemini no cabeçalho:

Sua chave é mantida localmente no seu navegador e enviada diretamente para a API oficial do Google.
