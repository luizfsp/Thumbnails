Como Funciona a IA?

Nós incluímos um sistema de **Tradução Localizada de Demonstração** para que você possa testar o aplicativo e ver o grid ser traduzido em tempo real para os 16 países imediatamente, mesmo sem uma chave de API.

Ao inserir sua Chave de API do Gemini no cabeçalho:

  • Você pode digitar qualquer frase personalizada em português (ex: "Estratégia Secreta Nunca Revelada").
  • O modelo gemini-2.5-flash-preview-09-2025 traduzirá dinamicamente com localização de gírias e tamanho ideal de termos para os 16 países!

Sua chave é mantida localmente no seu navegador e enviada diretamente para a API oficial do Google de forma segura.
